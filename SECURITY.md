# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Windows 10 | :white_check_mark: |
| Mac  | :x:|
| Linux  | :x:|

## Reporting a Vulnerability

Si votre version n'est pas prise en compte ou trop veille vous risquez d'avoir des bugs durant vos parties ou que le jeu crash.
